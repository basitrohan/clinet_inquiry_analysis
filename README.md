# Customer Satisfaction & Support Performance Trends

An operational dashboard focused on customer support efficiency, Service Level Agreement (SLA) compliance, and global feedback monitoring. This report provides a detailed view of ticket volume, resolution speeds, and regional performance flags.

## 📈 Support Key Performance Indicators (KPIs)

The dashboard tracks five essential metrics for support team health:

* **Total Tickets:** 5000
* **Tickets Met (SLA Compliance):** 4445
* **Total Raised Flags:** 555
* **Avg Resolution Time:** 942.82 Mins
* **SLA Breach Rate %:** 11.10%

## 📊 Visualizations & Operational Insights

1.  **Total Flags Raised by Month:** A time-series area chart tracking monthly escalations from January to September, identifying a significant peak in **August**.
2.  **Raised Flags by Region:** A global map visualization highlighting support hotspots across **North America (East/West), South America (Central), Europe (North), and Africa (South)**.
3.  **Hourly SLA Breach Rate %:** A treemap (heat map) breakdown visualizing breach frequency by the hour to identify peak pressure times on the support staff.
4.  **SLA Breach Rate % by Category:** A waterfall/bar analysis identifying the primary drivers of delays, with **Delay (26%)** and **KYC/Documentation (15%)** being the top contributors.
5.  **Multi-Channel Support Tracking:** Analysis segmented by support channels: **Chat, Email, Mobile App, and Phone**.

## 🤖 Data Generation
The data used to build this dashboard is **fully synthetic and AI-generated**.
* **Methodology:** Generated using a custom Python script to simulate customer support ticketing systems and global telemetry data.
* **Purpose:** This project is a portfolio demonstration showcasing expertise in operational analytics, CRM data modeling, and global performance tracking while maintaining data privacy.

* **Visualization Tool:** Power BI
* **Data Processing:** Python (Pandas) , SQL
* **Design Theme:** Professional Clean Blue aesthetic with high-contrast KPI cards.
